# Hola Mundo

Este es un proyecto simple que muestra "Hola Mundo" en una página web.

## Código

```html
<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Hola Mundo</title>
</head>
<body>
    <h1>Hola Mundo</h1>

    <!-- Sección de lo que te gusta -->
    <h2>Lo que me gusta:</h2>
    <ul>
        <li>Física ⚛️</li>
        <li>Programación 💻</li>
        <li>El asado 🥩</li>
        <li>Aeromodelismo a radio control 🛩️</li>
    </ul>

    <!-- Sección de lo que no te gusta -->
    <h2>Lo que no me gusta:</h2>
    <ul>
        <li>Los exámenes 📝</li>
        <li>El desorden 🗑️</li>
    </ul>
</body>
</html>
